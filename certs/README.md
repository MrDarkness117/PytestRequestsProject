# SSL Сертификаты

Положите сюда SSL сертификат для GigaChat API.

## Использование

1. Получите SSL сертификат от Сбербанка (`.pem` файл)
2. Переименуйте его в `sber_ca_bundle.pem` и положите в эту директорию
3. Установите переменную окружения `GIGACHAT_CA_BUNDLE=/certs/sber_ca_bundle.pem`

## Важно

⚠️ **Не коммитьте сертификаты в Git!** Файлы `.pem`, `.crt`, `.key` автоматически игнорируются через `.gitignore`.
